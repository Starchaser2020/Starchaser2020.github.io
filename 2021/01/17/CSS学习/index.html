CSS简单笔记

Jelon 发表于 2021-01-17

CSS是什么？

CSS是Cascading Sstyle Ssheets的简称，中文译为层叠样式表。HTML标签确定元素是什么类型，CSS则确定元素怎么显示。

CSS长什么样？

两部分构成：选择器+多条语句
selector {declaration1;declaration2...}

选择器

选择器类型

  • 派选择器：li strong{}选择li元素中的strong元素
  • id选择器：#id选择对应id元素，常常和派生选择器一起使用如#sidebar p{}选择id为sidebar元素中的p元素
  • 类选择器：.class选择属性相同的一类元素。常常和派生选择器一起使用如.class p{}选择属性为class元素中的p元素。
  • 属性选择器：[属性]、[属性=值]、[属性~=值]…，通常应用于表格中。

    CSS基本属性和值

  1. CSS背景
  • 简写属性background：可以设置所有background属性。用法body{background:#00FF00 url() no-repeat fixed top;}
  • background-attachment：设置背景图像是否固定；
  • background-color：设置元素的背景颜色；
  • background-image：将图像设置为背景；
  • background-position：设置背景图像的起始位置；
  • background-repeat：设置背景图像是否以及如何重复。
  1. CSS文本属性
    改变文本颜色、字符间距、对齐文本、装饰文本、文本缩进等

  • 缩进文本：p{text-indent: 5em;}-块级元素第一行缩进5em

    • 使用负值可以造成悬挂缩进的效果；
    • 使用百分比值：相对于父级元素的百分比；
    • 继承：缩进可以继承。

  • 水平对齐text-align
    水平对齐只是改变元素内部文本的排列并没有改变元素本身的位置。有以下几种对齐方式：left、right、center、justify、inherit

  • 字间隔word-spacing：调增字（词）间的间隔

  • 字母间隔letter-spacing：调整字母的间隔

  • 字符转换text-transform：顾名思义

    • none
    • uppercase
    • lowercase
    • capitalize

  • 文本装饰text-decoration：

    • none
    • underline
    • overline
    • line-through
    • blink

  • 处理空白符white-space

    • normal：删除多余空格和换行符变为空格；
    • pre：保留所有空格和换行符
    • nowrap：类似normal
    • pre-wrap：保留所有空格和换行符
    • pre-line：合并空格，保留换行符

  • 文本方向dir

    • ltr：从左到右；
    • rtl：从右到左。
  1. CSS字体font
    各种字体是千奇百怪，CSS规范了5种通用字体，两个方面考虑①不同系统通用性②减少工作量
    属性
  • font-family：规定字体类型
    • Serif
    • Sans-serif
    • Monospace
    • Cursive
    • Fantasy
  • font-style:规定斜体文本
    • normal 正常显示
    • italic 斜体显示
    • oblique 斜体显示
      italic和oblique的区别在于italic是字体的倾斜版本，而oblique则只是将字体倾斜。
  • font-variant：小型大写字母
  • font-weight：设置文本的粗细
  • font-size：设置文本大小
  1. CSS链接
    可以根据链接的状体来设置CSS：
  • a:link-普通的、未被访问的链接
  • a:visited-用户已访问的链接
  • a:hover-鼠标指针位于链接的上方
  • a:active-链接被点击的时刻
  1. CSS列表
  • list-style-type：列表项的符号
  • list-style-image：列表项的图像
    通常使用image时必须跟上一个type，以免image失效。
  1. CSS表格
  • border：边框
  • width：宽度
  • height：高度
  • text-align：文本对齐
  • vertical-align：垂直对齐
  • padding：内距
  • color：颜色
  1. CSS轮廓outline

    CSS框模型

CSS 框模型 (Box Model) 规定了元素框处理元素内容、内边距、边框 和 外边距 的方式

    CSS定位

    文档流：文档内元素的正常位置
    position属性的五个值：
  • static 定位：正常定位，遵循正常的文档流对象；
  • 相对定位：相对于文档流的定位，因为仍然是文档流，所以会覆盖其他框；
  • 绝对定位：相对于父元素或者html元素的定位，其原本文档流被删除，重新生成框，这个框不属于文档流，因此虽然覆盖其他元素，但不会占用文档流空间；
  • fixed定位：元素位置相对于浏览器窗口是固定位置；

    CSS伪类和伪元素

    CSS伪类

    是什么？一种特殊的表示某个元素状态的关键字。例如a:hover{color:red}表示用户hover任何链接时，链接呈现红色。
  • 语言类伪类
    • :dir
    • :lang
  • 定位伪类
    • :any-link——链接处于未被点击或者已被点击状态
    • :link
    • :visited
    • :local-link——匹配定位到当前页面的链接分为（0）（1）（2）…
    • :target
    • :target-within
    • :scope
  • 用户行为伪类
    • :hover
    • :active
    • :focus
    • :focus-visible
    • :focus-within
  • 时间维度的伪类（webvtt）
    • :current——目前现实的元素；
    • :past——:current之前的元素；
    • :future——:current之后的元素。
  • 资源状态伪类（音乐播放器、视频播放器）
    • :playing——媒体播放状态；
    • paused——媒体暂停状态；
  • 输入伪类（与表单元素的状态有关）
    • :enabled
    • :disabled
    • :read-only
    • :read-write
    • :placeholder-shown
    • :default——匹配默认值的表单项，参考default条件
    • checked——匹配被选过的元素；
    • :indeterminate——匹配不确定的元素，需要配合script使用；
    • :blank——匹配一个空输入或者null；
    • :valid——匹配输入是正确的元素；
    • :invalid——匹配输入是非法值得元素；
    • :in-range——匹配值在范围内得；
    • :out-of-range——匹配在范围外得；
    • :required——匹配设置required属性得元素；
    • :optional——匹配没有设置required属性得元素；
  • 树结构类（与元素在文档树的位置有关）
    • :root——匹配根元素，通常是<html>；
    • :empty——匹配没任何后代的元素；
    • nth-child
    • nth-last-child
    • :first-child
    • :last-child
    • :only-child
    • nth-of-type
    • nth-last-of-type
    • :first-of-type
    • :last-of-type
    • :only-of-type
  • 总结：伪类基于不同状态的的元素进行选择，使页面丰富。
