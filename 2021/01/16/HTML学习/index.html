HTML简单笔记

Jelon 发表于 2021-01-16

HTML简介

  1. HTML(Hyper Text Markup Language)

    • HTML是一种标记语言（何为标记？标记其他语言的状态：属性、位置等等）

  2. HTML标签

    • HTML 标记标签通常被称为 HTML 标签 (HTML tag)。
    • HTML 标签是由尖括号包围的关键词，比如 <html>
    • HTML 标签通常是成对出现的，比如 <b> 和 </b>标签对中的第一个标签是开始标签，第二个标签是结束标签开始和结束标签也被称为开放标签和闭合标签

  3. HTML 文档 = 网页
    简单来说一个HTML文档即一个打开的网页。
    基本元素：

    • <html>与</html> 之间的文本描述网页
    • <body>与 </body> 之间的文本是可见的页面内容
    • <h1> 与 </h1>之间的文本被显示为标题
    • <p> 与 </p> 之间的文本被显示为段落

  4. HTML元素
    HTML元素是从开始标签到闭合标签的所有内容代码内容

  5. HTML属性
    HTML属性为元素提供附加信息。
    HTML标签可以拥有属性，即尖括号内可以拥有属性，属性以*名字：”值”的方式出现，例如：name=”value”，属性值都是包括在引号内的。

    标签

    注释标签：<!-- this is a comment -->
    水平线：<hr />
    段落：<p>这里添加内容</p>
    样式：准确来讲，样式可以是一个属性，定义标签的显示，比如<h1 style = "font-family:verdana">A heading</h1>
    文本格式化标签：<pre></pre>-预格式化标签，保留空格和换行…
    双向重写：<bdo></bdo>
    样式：

  • 外部样式- <head><link rel="stylesheet" type="text/css" href="mystyle.css"></head>
  • 内部样式表（类似于标签）-
    <head><style type="text/css">body {background-color: red}p {margin-left: 20px}</style></head>
  • 内部样式：其他标签属性

链接：两种链接

  • 超文本链接：<a href="www.baidu.com"></a>
  • 锚链接：一个前提有一个锚<a id = 'tips'></a>
                  访问这个锚`<a href='www.baidu.com#tips'></a>`

图像：<img src="url" />
表格：<table>表</table>、<tr>行</tr>、<tb>列</tb>
列表

  • 无序列表：<ul><li></li></ul>
  • 有序列表：<ol><li></li></ol>
  • 定义列表：

    <dl>
    <dt>Coffee</dt>
    <dd>Black hot drink</dd>
    <dt>Milk</dt>
    <dd>White cold drink</dd>
    </dl>

    元素分类
    • 块元素(block level element)
      通常会以新行来开始（和结束）<h1>, <p>, <ul>, <table>
      • div是块级元素，它是可用于组合其他 HTML 元素的容器。<div> 元素没有特定的含义。除此之外，由于它属于块级元素，浏览器会在其前后显示折行。作用：
        1. 对大的内容块设置样式属性；
        2. 文档布局。
    • 内联元素(inline element)
      通常不会以新行开始<b>, <td>, <a>, <img>
      • HTML <span> 元素是内联元素，可用作文本的容器。<span> 元素也没有特定的含义。当与 CSS 一同使用时， 元素可用于为部分文本设置样式属性。
