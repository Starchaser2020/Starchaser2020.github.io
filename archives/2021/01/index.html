CSS简单笔记

Jelon 发表于 2021-01-17    标签： CSS

1. CSS是什么？

CSS是Cascading Sstyle Ssheets的简称，中文译为层叠样式表。HTML标签确定元素是什么类型，CSS则确定元素怎么显示。

2. CSS长什么样？

两部分构成：选择器{多条语句}
selector {declaration1;declaration2...}

3. 选择器

  • 派选择器：li strong{}选择li元素中的strong元素
  • id选择器：#id选择对应id元素，常常和派生选择器一起使用如#sidebar p{}选择id为sidebar元素中的p元素
  • 类选择器：.class选择属性相同的一类元素。常常和派生选择器一起使用如.class p{}选择属性为class元素中的p元素。
  • 属性选择器：[属性]、[属性=值]、[属性~=值]…，通常应用于表格中。

4. CSS基本属性和值

CSS背景

  • 简写属性background：可以设置所有background属性。用法body{background:#00FF00 url() no*repeat fixed top;}
  • background-attachment：设置背景图像是否固定；
  • background-color：设置元素的背景颜色；
  • background-image：将图像设置为背景；
  • background-position：设置背景图像的起始位置；
  • background-repeat：设置背景图像是否以及如何重复。

CSS文本属性

改变文本颜色、字符间距、对齐文本、装饰文本、文本缩进等

  • 缩进文本：p{text-indent: 5em;}-块级元素第一行缩进5em

    • 使用负值可以造成悬挂缩进的效果；
    • 使用百分比值：相对于父级元素的百分比；
    • 继承：缩进可以继承。

  • 水平对齐text-align

    水平对齐只是改变元素内部文本的排列并没有改变元素本身的位置。有以下几种对齐方式：left、right、center、justify、inherit

  • 字间隔word-spacing：调增字（词）间的间隔

  • 字母间隔letter-spacing：调整字母的间隔

  • 字符转换text-transform：顾名思义

    • none
    • uppercase
    • lowercase
    • capitalize

  • 文本装饰text-decoration：

    • none
    • underline
    • overline
    • line-through
    • blink

  • 处理空白符white-space

    • normal：删除多余空格和换行符变为空格；
    • pre：保留所有空格和换行符
    • nowrap：类似normal
    • pre-wrap：保留所有空格和换行符
    • pre-line：合并空格，保留换行符

  • 文本方向dir

    • ltr：从左到右；
    • rtl：从右到左。

CSS字体font

各种字体是千奇百怪，CSS规范了5种通用字体，两个方面考虑①不同系统通用性②减少工作量

  • font-family：规定字体类型

    • Serif
    • Sans-serif
    • Moonscape
    • Cursive
    • Fantasy

  • font-style: 规定斜体文本

    • normal 正常显示
    • italic 斜体显示
    • oblique 斜体显示

    italic和oblique的区别在于italic是字体的倾斜版本，而oblique则只是将字体倾斜。

  • font-variant：小型大写字母

  • font-weight：设置文本的粗细

  • font-size：设置文本大小

CSS链接

可以根据链接的状体来设置CSS：

  • a:link-普通的、未被访问的链接
  • a:visited-用户已访问的链接
  • a:hover-鼠标指针位于链接的上方
  • a:active-链接被点击的时刻

CSS列表

  • list-style-type：列表项的符号

  • list-style-image：列表项的图像

    通常使用image时必须跟上一个type，以免image失效。

CSS表格

  • border：边框
  • width：宽度
  • height：高度
  • text-align：文本对齐
  • vertical-align：垂直对齐
  • padding：内距
  • color：颜色

6. CSS定位

文档流：文档内元素的正常位置

position属性的五个值：

  • static 定位：正常定位，遵循正常的文档流对象；
  • 相对定位：相对于文档流的定位，因为仍然是文档流，所以会覆盖其他框；
  • 绝对定位：相对于父元素或者html元素的定位，其原本文档流被删除，重新生成框，这个框不属于文档流，因此虽然覆盖其他元素，但不会占用文档流空间；
  • fixed定位：元素位置相对于浏览器窗口是固定位置；

8. CSS伪类和伪元素

1. CSS伪类

是什么？一种特殊的表示某个元素状态的关键字。例如a:hover{color:red}表示用户hover任何链接时，链接呈现红色。

  • 语言类伪类

    • :dir
    • :lang

  • 定位伪类

    • :any-link——链接处于未被点击或者已被点击状态
    • :link
    • :visited
    • :loca-link——匹配定位到当前页面的链接分为（0）（1）（2）…
    • :target
    • :target-within
    • :scope

  • 用户行为伪类

    • :hover
    • :active
    • :focus
    • :focus-visible
    • :focus-within

  • 时间维度的伪类（webvtt）

    • :current——目前现实的元素；
    • :past——:current之前的元素；
    • :future——:current之后的元素。

  • 资源状态伪类（音乐播放器、视频播放器）

    • :playing——媒体播放状态；
    • paused——媒体暂停状态；

  • 输入伪类（与表单元素的状态有关）

    • :enabled
    • :disabled
    • :read-only
    • :read-write
    • :placeholder-shown
    • :default——匹配默认值的表单项，参考default条件
    • checked——匹配被选过的元素；
    • :indeterminate——匹配不确定的元素，需要配合script使用；
    • :blank——匹配一个空输入或者null；
    • :valid——匹配输入是正确的元素；
    • :invalid——匹配输入是非法值得元素；
    • :in-range——匹配值在范围内得；
    • :out-of-range——匹配在范围外得；
    • :required——匹配设置required属性得元素；
    • :optional——匹配没有设置required属性得元素；

  • 树结构类（与元素在文档树的位置有关）

    • :root——匹配根元素，通常是<html>；
    • :empty——匹配没任何后代的元素；
    • nth-child
    • nth-last-child
    • :first-child
    • :last-child
    • :only-child
    • nth-of-type
    • nth-last-of-type
    • :first-of-type
    • :last-of-type
    • :only-of-type

  • 总结：伪类基于不同状态的的元素进行选择，使页面丰富。

  • ::after——前者元素中添加子元素；

HTML简单笔记

Jelon 发表于 2021-01-16    标签： HTML

HTML简介

  1. HTML(Hyper Text Markup Language)

    • HTML是一种标记语言（何为标记？标记其他语言的状态：属性、位置等等）

  2. HTML标签

    • HTML 标记标签通常被称为 HTML 标签 (HTML tag)。
    • HTML 标签是由尖括号包围的关键词，比如 <html>
    • HTML 标签通常是成对出现的，比如 <b> 和 </b>标签对中的第一个标签是开始标签，第二个标签是结束标签开始和结束标签也被称为开放标签和闭合标签

  3. HTML 文档 = 网页
    简单来说一个HTML文档即一个打开的网页。
    基本元素：

    • <html>与</html> 之间的文本描述网页
    • <body>与 </body> 之间的文本是可见的页面内容
    • <h1> 与 </h1>之间的文本被显示为标题
    • <p> 与 </p> 之间的文本被显示为段落

  4. HTML元素
    HTML元素是从开始标签到闭合标签的所有内容代码内容

  5. HTML属性
    HTML属性为元素提供附加信息。
    HTML标签可以拥有属性，即尖括号内可以拥有属性，属性以*名字：”值”的方式出现，例如：name=”value”，属性值都是包括在引号内的。

    标签

    注释标签：<!-- this is a comment -->
    水平线：<hr />
    段落：<p>这里添加内容</p>
    样式：准确来讲，样式可以是一个属性，定义标签的显示，比如<h1 style = "font-family:verdana">A heading</h1>
    文本格式化标签：<pre></pre>-预格式化标签，保留空格和换行…
    双向重写：<bdo></bdo>
    样式：

  • 外部样式- <head><link rel="stylesheet" type="text/css" href="mystyle.css"></head>
  • 内部样式表（类似于标签）-
    <head><style type="text/css">body {background-color: red}p {margin-left: 20px}</style></head>
  • 内部样式：其他标签属性

链接：两种链接

  • 超文本链接：<a href="www.baidu.com"></a>
  • 锚链接：一个前提有一个锚<a id = 'tips'></a>
                  访问这个锚`<a href='www.baidu.com#tips'></a>`

图像：<img src="url" />
表格：<table>表</table>、<tr>行</tr>、<tb>列</tb>
列表

  • 无序列表：<ul><li></li></ul>
  • 有序列表：<ol><li></li></ol>
  • 定义列表：

    <dl>
    <dt>Coffee</dt>
    <dd>Black hot drink</dd>
    <dt>Milk</dt>
    <dd>White cold drink</dd>
    </dl>

元素分类

  • 块元素(block level element)
      通常会以新行来开始（和结束）`<h1>, <p>, <ul>, <table>`
    • div是块级元素，它是可用于组合其他 HTML 元素的容器。<div> 元素没有特定的含义。除此之外，由于它属于块级元素，浏览器会在其前后显示折行。作用：
        1. 对大的内容块设置样式属性；
  2. 文档布局。
    • 内联元素(inline element)
      通常不会以新行开始<b>, <td>, <a>, <img>
      • HTML <span> 元素是内联元素，可用作文本的容器。<span> 元素也没有特定的含义。当与 CSS 一同使用时， 元素可用于为部分文本设置样式属性。
