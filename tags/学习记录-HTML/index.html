HTML简介

  1. HTML(Hyper Text Markup Language)

    • HTML是一种标记语言（何为标记？标记其他语言的状态：属性、位置等等）

  2. HTML标签

    • HTML 标记标签通常被称为 HTML 标签 (HTML tag)。
    • HTML 标签是由尖括号包围的关键词，比如 <html>
    • HTML 标签通常是成对出现的，比如 <b> 和 </b>标签对中的第一个标签是开始标签，第二个标签是结束标签开始和结束标签也被称为开放标签和闭合标签

  3. HTML 文档 = 网页
    简单来说一个HTML文档即一个打开的网页。
    基本元素：

    • <html>与</html> 之间的文本描述网页
    • <body>与 </body> 之间的文本是可见的页面内容
    • <h1> 与 </h1>之间的文本被显示为标题
    • <p> 与 </p> 之间的文本被显示为段落

  4. HTML元素
    HTML元素是从开始标签到闭合标签的所有内容代码内容

  5. HTML属性
    HTML属性为元素提供附加信息。
    HTML标签可以拥有属性，即尖括号内可以拥有属性，属性以*名字：”值”的方式出现，例如：name=”value”，属性值都是包括在引号内的。

    标签

    注释标签：<!-- this is a comment -->
    水平线：<hr />
    段落：<p>这里添加内容</p>
    样式：<style></style>
